# Homelab Documentation

## Descripción
Documentación de mi entorno de laboratorio personal para aprendizaje de TI y ciberseguridad.

## Infraestructura
- Proxmox VE 9.1.1 corriendo en Lenovo Legion (16GB RAM, 1TB SSD)
- VM Ubuntu 26.04 Desktop
- SSH configurado y funcionando

## Lo que aprendí
- Instalación y configuración de Proxmox
- Creación de máquinas virtuales
- Instalación de Ubuntu en VM
- Configuración de SSH
- Conexión remota entre máquinas
