# osTicket v1.18.1 — Instalación en Homelab

## Descripción
Sistema de ticketing de soporte técnico instalado en VM Ubuntu 25.10 sobre Proxmox VE 9.1.1.

## Stack técnico
- OS: Ubuntu 25.10
- Web server: Apache2
- Base de datos: MariaDB
- Lenguaje: PHP 8.4
- Herramienta: osTicket v1.18.1

## Pasos realizados
1. Instalación de Apache, MariaDB y PHP
2. Creación de base de datos y usuario en MariaDB
3. Descarga y extracción de osTicket v1.18.1
4. Configuración de permisos de archivos
5. Instalación via interfaz web
6. Simulación de flujo completo: creación de ticket → atención → cierre
